Två frågor om CSS och koppling mellan HTML och CSS med id och class.

  1. Fråga E1: Vad menas med kaskad i CSS? Hur fungerar det?
  2. Fråga D2: I HTML-taggarna kan man ha attribut med id och class och sedan refera till dem i CSS-koden. Vad är skillnaden, när ska man använda id och när ska man använda class.

Mitt Svar

E1

Kaskad innebär att flera stilar kan appliceras till en specifk del av HTML taggarna. T.ex så kan en stilregel applicera färgen röd på H3 taggar medan det finns en annan stilregel som vill att H3 taggarna ska vara gröna.
Det finns därför generela regler och specifika regler. Den mest specifika regeln appliceras i CSS.

E2

Id är unikt, varje HTML element kan enbart ha ett ID och varje HTML-sida kan bara ha ett element med det ID:et.
Class däremot är inte unikt. Det går att använda samma class på flera element och det går att använda flera classes på samma element.

ID bör användas när man har ett element på en sida som ska få en stil applicerad. Exempel på element som man brukar använda id på är: header, main-contents, footer eller sidebars.

Class bör användas när man vill konsekvent applicera stilar på flera element på flera sidor.
